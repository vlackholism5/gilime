# v1.4-05 Smoke Checklist

PC에서 DDL 실행 후 아래 항목을 확인하세요.

## 1) /public/user/home.php loads

- URL: `http://localhost/gilime_mvp_01/user/home.php` (또는 APP_BASE/user/home.php)
- 홈 페이지가 로드되고, 네비(Home / Routes / Alerts)와 "My subscriptions: N" 표시.

## 2) /public/user/routes.php loads, subscribe works

- URL: `http://localhost/gilime_mvp_01/user/routes.php`
- 노선 목록(doc_id, route_label) 표시. Subscribe 버튼 클릭 시 구독 등록, Unsubscribe 시 해제. 새로고침 시 상태 유지.

## 3) /public/user/alerts.php loads

- URL: `http://localhost/gilime_mvp_01/user/alerts.php`
- 알림 목록 표시(없으면 "알림이 없습니다"). type=strike / event / update 필터 동작.

## 4) run_alert_ingest_stub inserts events

- 프로젝트 루트에서: `php scripts/php/run_alert_ingest_stub.php`
- Windows(XAMPP)에서 php가 PATH에 없으면: `C:\xampp\php\php.exe scripts\run_alert_ingest_stub.php`
- 출력: "run_alert_ingest_stub: inserted N dummy event(s)." (최초 3, 재실행 시 0)
- alerts.php / home.php에서 더미 알림 표시 확인.

## 5) Admin pages unchanged

- /admin/index.php, /admin/review_queue.php, /admin/route_review.php 등 기존 관리자 플로우 동작. SoT/게이트/매칭 로직 변경 없음.
