## 체크 항목
- 공통 스타일 파일 `public/assets/css/gilaime_ui.css`가 로드된다.
- 상단 네비게이션, 카드, 표 헤더/행 hover가 통일된 톤으로 보인다.
- 주요 상태값(초안/발행/실패 등)이 배지 형태로 구분된다.
- 사용자/관리자 주요 CTA 버튼 문구가 한국어 중심으로 표시된다.
- 큰 표에서 가로 스크롤 없이도 기본 사용이 가능하다.

## 수동 검증
1. 관리자 로그인 후 문서 허브 진입.
2. 문서 상세에서 파싱 실행 버튼/상태 섹션 확인.
3. 알림 운영에서 초안 생성, 발행 버튼 표시 확인.
4. 사용자 홈/노선/알림 페이지에서 한국어 카피와 버튼 상태 확인.
