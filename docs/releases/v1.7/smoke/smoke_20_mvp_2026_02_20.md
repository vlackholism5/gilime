# smoke_20_mvp_2026_02_20

## 목적
- 2/20 MVP 기준(업로드→파싱→검수→승격→사용자 경로안내)을 수동으로 점검한다.

## 사전 조건
- 관리자 계정으로 로그인 가능
- 테스트용 PDF 1건 준비
- 사용자 계정에 노선 구독 1건 이상 존재

## 시나리오 A: 완주
1. `admin/upload_pdf.php`에서 PDF 업로드
2. 업로드 결과의 문서 상세 진입
3. `파싱/매칭 실행` 수행
4. `route_review`에서 후보 승인/거절 반영
5. `승인 후보를 Route Stops로 승격` 수행
6. `user/journey.php`에서 같은 노선의 정류장 순서 표시 확인

### 기대 결과
- 문서 상세에서 성공 플래시 확인
- route_review에서 후보 상태 변경 반영
- user 경로안내에서 정류장 목록 표시

## 시나리오 B: 실패/복구
1. 잘못된 파일 경로 또는 비정상 PDF로 파싱 시도
2. 문서 상세에서 실패 안내/오류 코드 확인
3. 정상 PDF로 재실행 후 성공 확인

### 기대 결과
- 실패 시 경고 안내가 표시됨
- 재실행 후 성공 흐름으로 복귀됨
