# v1.7 E2E

## 시나리오 A: 완주(성공) 시나리오
1. 관리자에서 PDF 업로드
2. 문서 상세에서 파싱/매칭 실행
3. 노선 검수에서 승인/거절/별칭 정리
4. 승격(PROMOTE) 실행
5. 사용자 경로안내에서 정류장 순서 확인
6. 알림 감사/운영 화면에서 전달 이력 확인

### 기대 결과
- 후보/지표/승격 결과가 일관되게 표시된다.
- 사용자 화면에서 최신 승격 노선을 조회할 수 있다.
- 마이노선 → 경로 안내 링크로 journey 정류장 순서 확인 가능.

## 시나리오 B: 실패(예외) 시나리오
1. 잘못된 PDF 또는 경로 오류로 파싱 실패 유도
2. 문서 상세에서 오류 코드/실패 안내 확인
3. 원인 수정 후 재실행

### 기대 결과
- 실패 원인이 플래시/메타에 표시된다.
- 재실행 후 성공 시 정상 흐름으로 복귀한다.
