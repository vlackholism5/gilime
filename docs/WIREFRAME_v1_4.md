## 2. Routes (/user/routes.php)

```
+------------------------------------------------------------------+
|  GILIME                    [홈] [내 구독] [알림] [로그아웃]       |
+------------------------------------------------------------------+
|  노선 목록                                                        |
|  -----------------------------------------------------------------|
|  [노선명/번호 검색_____________] [검색]  [정류장순 정렬 v]         |
|                                                                  |
|  +--------------------------------------------------------------+|
|  | 노선명(라벨)  | 구간 요약           | 정류장 수 | [보기] [구독]  ||
|  | 101          | A정류장 ~ B정류장   | 24       | [보기] [구독]  ||
|  | 102          | B정류장 ~ C정류장   | 18       | [보기] [구독]  ||
|  +--------------------------------------------------------------+|
|  (promote된 route_stop 기준 조회)                                 |
+------------------------------------------------------------------+
```
