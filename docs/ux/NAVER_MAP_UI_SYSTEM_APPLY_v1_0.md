## 지도 렌더링 시점

- **지도(폴리라인 포함)는 Phase-2에서만 렌더링한다.** MVP와 MVP+에서는 지도 컴포넌트를 노출하지 않는다.

---

## MVP/MVP+ "지도 없이도 UX 무너지지 않음" 규칙

1. **완결 요소:** 출발/도착 입력 + 경로 찾기 + 경로 목록(텍스트/카드)만으로 흐름이 완결된다. 사용자는 지도 없이도 출발지·도착지 선택과 경로 결과 확인이 가능해야 한다.
2. **"지도 보기" 노출:** "지도 보기" 버튼 또는 탭은 Phase-2 전까지 비노출 또는 비활성 처리한다. 노출 시에는 "Phase-2에서 제공" 등 문구로만 안내한다.
3. **경로 상세 표시:** 경로 상세에서는 정류장 순서, 노선명, 예상 소요 시간만 표시한다. 지도 위 폴리라인·마커는 Phase-2에서 추가한다.

이 규칙으로 MVP/MVP+ 단계에서도 UI 일관성이 유지되고, Phase-2에서 지도가 추가되어도 기존 플로우가 깨지지 않도록 한다.
