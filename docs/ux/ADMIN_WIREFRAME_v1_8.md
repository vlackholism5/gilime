# 관리자 화면 와이어프레임 (v1.8)

## 1. 정보 구조 (IA)

```
[로그인]
    ↓
[문서 허브] (index.php)
    ├── PDF 업로드 (upload_pdf.php)
    ├── 문서 상세 (doc.php?id=N)
    │   └── 노선 검수 (route_review.php?source_doc_id=N&route_label=R)
    └── 검수 대기열 (review_queue.php)

[운영] (ops_dashboard.php)
    ├── 운영 요약 (ops_summary.php)
    └── 운영 제어 (ops_control.php)

[알림] (alert_ops.php)
    └── 알림 감사 (alert_event_audit.php)

[감사] (alias_audit.php)
```

---

## 2. 공통 레이아웃 (실무형)

```
+------------------------------------------+
| 문서 | 운영 | 알림 | 감사          [로그아웃] |
+------------------------------------------+
| breadcrumb: 문서 허브 / 현재 페이지       |
+------------------------------------------+
| [페이지 제목]                            |
| [헬퍼 텍스트]                            |
+------------------------------------------+
| [액션 버튼(선택)]                        |
+------------------------------------------+
| [본문: 카드/테이블/폼]                   |
+------------------------------------------+
| [단축키 안내] details.kbd-help (선택)    |
+------------------------------------------+
```

- **상단:** `render_admin_nav()` — 그룹 링크 + 로그아웃
- **Breadcrumb:** `render_admin_header($breadcrumbs)` — 문서 허브 / 현재 페이지
- **본문:** `g-page-head` + `card g-card` + `g-table`
- **8pt 그리드:** `--g-space-1` ~ `--g-space-4` (NAVER_MAP_GRID_GUIDE)

## 3. 화면별 ASCII 와이어프레임

### 3.1 문서 허브 (index.php)

```
+------------------------------------------+
| 문서 | 운영 | 알림 | 감사          [로그아웃] |
+------------------------------------------+
| 문서 허브                                 |
+------------------------------------------+
| 문서 허브                                 |
| 문서 상태를 확인하고 검수/운영 화면으로 이동 |
| [실패건][전체][검수대기][Alias][알림]...   |
+------------------------------------------+
| [단축키 안내]                             |
+------------------------------------------+
| ID | 출처 | 제목 | ... | 파싱·매칭 현황 | 노선검수 |
|----|------|------|-----|----------------|----------|
| 1  | ...  | ...  | ... | [파싱·매칭]    | [노선검수 R1] |
+------------------------------------------+
```

### 3.2 문서 상세 (doc.php)

```
+------------------------------------------+
| 문서 | 운영 | 알림 | 감사          [로그아웃] |
+------------------------------------------+
| 문서 허브 / 문서 #10                      |
+------------------------------------------+
| 문서 파싱 · 노선 매칭 현황               |
| 문서 #10 · 상태 확인, 파싱/매칭 실행      |
+------------------------------------------+
| [메타] 출처, 제목, OCR, 파싱 상태...      |
| [파싱 실행] [매칭 실행] [노선 검수]        |
+------------------------------------------+
| [테이블] Jobs, Routes, 후보...            |
+------------------------------------------+
```

### 3.3 노선 검수 (route_review.php)
