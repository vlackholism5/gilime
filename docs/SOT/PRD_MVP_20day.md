# SoT — PRD MVP 20일

## Purpose

Single source of truth for MVP 범위·문제 정의·포함/제외·KPI. 20일 단위 기준.

**Lock 조건:** Scope·KPI·Non-goals 고정, PRD 리뷰 통과.

---

## 제품 정의

교통 이슈 시 임시셔틀 포함 대체 경로 제공. 이슈 기반 재탐색 및 구독 경로 안내.

---

## 문제 정의

- 지하철 중단 시 대체 경로 혼란
- 셔틀 정보 분산
- 실시간 이슈 반영 부족

---

## 포함 기능 (MVP)

- 출발/도착 검색
- 이슈 표시
- 임시셔틀 포함 경로
- 정렬: 최적/최소시간/최소환승
- 구독
- 안내 모드

---

## 제외 (MVP+)

- AI 예측
- 실시간 혼잡도
- 전국 확대

---

## KPI

| 지표 | 목표 |
|------|------|
| 경로 응답 | ≤ 2초 |
| 재탐색 성공 | ≥ 95% |
| 구독 전환 | ≥ 15% |
| 안내 시작률 | 40% |

---

## MVP vs MVP+

- 자동 재탐색: MVP에서 부분 지원 → MVP+에서 실시간 강화
- 혼잡도·AI: MVP+ 범위
