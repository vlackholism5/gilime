# UI_COPY_KO

## 공통 액션 라벨
- Create -> 생성
- Publish -> 발행
- Review -> 검수
- Upload -> 업로드
- Run Parse/Match -> 파싱/매칭 실행
- Filter -> 필터 적용
- Back -> 뒤로
- Logout -> 로그아웃

## 상태 번역
- draft -> 초안
- published -> 발행
- pending -> 대기
- shown -> 표시됨
- sent -> 발송됨
- failed -> 실패

## 플래시 메시지 템플릿
- 생성 성공: `생성 완료`
- 발행 성공: `발행 완료`
- 큐 적재 성공: `발행 완료: 대상 {count}명 큐 적재`
- 권한 차단: `차단: 승인 권한이 없습니다`
- 대상 없음 차단: `차단: 대상 사용자가 없습니다(구독 조건 불일치)`
- 일반 실패: `실패: 처리 중 오류가 발생했습니다`
- 업로드 성공: `업로드 완료: {file} / source_doc_id={id}`
- 업로드 실패: `업로드 실패: {reason}`

## 문장 톤 가이드
- 짧은 문장, 원인 먼저
- 영문 혼용 최소화 (필요한 고유명만 유지)
- 운영 화면은 "다음 행동"이 보이도록 안내 문구 배치
